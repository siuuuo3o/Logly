# 🚀 개발 순서 가이드

> 혼자 개발 + 바이브 코딩 + 1~2개월 스토어 배포 목표 기준으로 작성했어요.
> AI 글쓰기 보조 기능은 모듈로 분리되어 있어 언제든 제거 가능해요.

---

## 📌 핵심 원칙

- **수직 개발** — 기능 하나를 백엔드부터 화면까지 완성하고 다음으로 넘어가기
- **배포 가능한 상태 유지** — 언제 멈춰도 돌아가는 앱이 있어야 함
- **AI 기능은 마지막** — 핵심 기능이 먼저, AI는 나중에 얹기

---

## 🗓️ Phase 1 — 기반 세팅 (1~3일)

> 개발 환경 세팅 + 프로젝트 뼈대 잡기

### Backend
- [ ] Spring Boot 프로젝트 생성 (Gradle)
- [ ] MySQL + JPA 연동
- [ ] 공통 응답 포맷 설정 (`ApiResponse<T>`)
- [ ] JWT 인증 필터 세팅
- [ ] S3 연동 설정 (이미지 업로드)
- [ ] Swagger 설정
- [ ] 환경변수 분리 (`application-local.yml`)

### Frontend
- [ ] Expo 프로젝트 생성 (TypeScript)
- [ ] React Navigation 세팅 (Stack + Tab)
- [ ] Axios 인스턴스 생성 + JWT 인터셉터
- [ ] Zustand 스토어 초기 구성
- [ ] 공통 컴포넌트 생성 (`Button`, `Input`, `Card`)
- [ ] 환경변수 설정 (`.env`)

---

## 🗓️ Phase 2 — 인증 (3~4일)

> 회원가입 / 로그인 / 소셜 로그인

### Backend
- [ ] `User` 엔티티 + 레포지토리
- [ ] 이메일 회원가입 API
- [ ] 로그인 + JWT 발급 API
- [ ] 토큰 재발급 API
- [ ] 카카오 소셜 로그인

### Frontend
- [ ] 스플래시 화면
- [ ] 온보딩 화면
- [ ] 로그인 화면
- [ ] 회원가입 화면
- [ ] 토큰 저장 + 자동 로그인 처리

---

## 🗓️ Phase 3 — 핵심: 기록 (5~7일)

> 앱의 가장 중요한 기능 — 사진 + 장소 + 일기 기록

### Backend
- [ ] `Place` 엔티티 + 장소 저장 API
- [ ] `Record` 엔티티 + CRUD API
- [ ] `RecordImage` 엔티티 + S3 이미지 업로드 API
- [ ] 타임라인 조회 API (날짜 정렬, 페이지네이션)
- [ ] OpenWeatherMap API 연동 (날씨 자동 저장)

### Frontend
- [ ] 기록하기 ① 화면 (사진 + 날짜 + 장소 + 날씨)
- [ ] 기록하기 ② 화면 (일기 + 공유 설정)
- [ ] Google Places API 연동 (장소 검색)
- [ ] 이미지 업로드 (S3)
- [ ] 홈 타임라인 화면
- [ ] 기록 상세 화면

---

## 🗓️ Phase 4 — 지도 (3~4일)

> 기록한 장소들을 지도에 시각화

### Backend
- [ ] 지도용 기록 목록 API (위도/경도 포함)
- [ ] 장소별 기록 클러스터링 API

### Frontend
- [ ] 지도 화면 (React Native Maps)
- [ ] 기록 핀 표시 (일상/여행 색 구분)
- [ ] 여행 코스 라인 연결
- [ ] 핀 클릭 → 기록 팝업 → 상세 이동

---

## 🗓️ Phase 5 — 여행 앨범 (4~5일)

> 여행 기간 묶기 + 앨범 뷰 + 일기 관리

### Backend
- [ ] `Trip` 엔티티 + CRUD API
- [ ] 여행-기록 연결 API
- [ ] 여행 상세 조회 API (사진 그리드 + 일기 목록 + 코스)

### Frontend
- [ ] 여행 앨범 목록 화면
- [ ] 여행 앨범 상세 화면 (탭: 사진 / 일기 / 지도 / 코스)
- [ ] 여행 생성 + 기간 설정
- [ ] 일기 작성 화면 (직접 작성)
- [ ] 일기 목록 + 상세 보기

---

## 🗓️ Phase 6 — 공유 (3~4일)

## 🗓️ Phase 7 — AI 글쓰기 보조 (2~3일) ⚠️ 선택

> 핵심 기능 완성 후 붙이기. 별로면 이 Phase 통째로 스킵 가능.

### Backend
- [ ] OpenAI GPT-4o API 연동

- [ ] AI 이어쓰기 API (`POST /api/v1/ai/assist/continue`)
- [ ] AI 제목 추천 API (`POST /api/v1/ai/assist/title`)
- [ ] `AiAssistLog` 로그 저장

### Frontend
- [ ] `AiWritingAssistant` 컴포넌트 생성 (독립 모듈)
- [ ] 일기 작성 화면에 컴포넌트 연결 (접고 펼치기)
- [ ] AI 결과 → 텍스트 편집창에 삽입

> 💡 **제거 방법**: `AiWritingAssistant` 컴포넌트 제거 + `/api/v1/ai/assist` 라우트 제거
